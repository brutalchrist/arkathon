Clon del mítico juego desarrollado por Taito en el año 1986, Arkanoid.

## Dependencies

- [PyGame](https://www.pygame.org/news)
  ```bash
  $ pip install pygame
  ```
  
## Clone

```bash
$ git clone https://github.com/brutalchrist/arkathon.git
$ cd arkathon
```

## Play

```bash
$ python arkathon.py
```
